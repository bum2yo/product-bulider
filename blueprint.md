# industry 5.0.1 - Introduction Page

## Overview
This project is a minimalist, single-page website with a terminal-like aesthetic for "industry 5.0.1". It features an interactive snow animation, a dark/light mode toggle, and a popup contact form integrating with Formspree.

## Features Implemented
- **Terminal UI:** Minimalist text layout with a blinking cursor.
- **Snow Animation:** Canvas-based interactive snowfall effect that adapts to the theme.
- **Theme Toggle:** Switch between Light (default) and Dark modes, persisted via local storage.
- **Contact Form:** Modal popup with a form for Name, Email, and Message, handling submissions via Formspree.
- **Hello Popup:** A popup that appears on page load with the message "안녕하세요".
- **Responsive Design:** Adjustments for mobile viewports.
- **Single File Structure:** All HTML, CSS, and JS contained within `index.html` for simplicity.
